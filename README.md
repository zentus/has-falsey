## has-falsey

Takes an array, and returns true if it contains one or more falsey items

### Installation
```bash
npm install --save has-falsey
```

### Usage

```javascript
import hasFalsey from 'has-falsey'

hasFalsey(['one', 'is', false])
//=> true

hasFalsey(['one', 'more', 'time'])
//=> false
```
